# 🚀 错误检查功能 - 快速开始

## 🎯 5秒了解

输入或处理任何 LaTeX 代码后，**错误检查器自动运行**，在输出框下方显示黄色警告面板。

## 🔴 常见错误一览

### 1. 括号不匹配 ← 最常见！
```latex
\frac{1}{2  ← 缺少 }
```
**修复：** `\frac{1}{2}` 补上缺失的右括号

---

### 2. 环境不闭合
```latex
\begin{equation}
x = 1
← 缺少 \end{equation}
```
**修复：** 在末尾添加 `\end{equation}`

---

### 3. $ 符号数量错误
```latex
$x = 1  ← 缺少右 $
```
**修复：** `$x = 1$` 补上右 `$`

---

### 4. 中文混在数学模式
```latex
$圆的面积 = πr²$  ← ❌ 中文不能这样用
```
**修复：** `圆的面积 = $πr²$` 把中文拿出来

---

## 🔴🟡🟠 错误等级

| 级别 | 图标 | 含义 | 处理 |
|------|------|------|------|
| 严重 | 🔴 | 会导致编译失败 | **必须修复** |
| 警告 | 🟡 | 可能有问题 | 最好修复 |
| 提示 | 🟠 | 潜在问题 | 建议关注 |

## 💡 实用技巧

### 技巧 1：从严重错误开始修复
优先处理红色 🔴 错误，再看黄色 🟡 错误

### 技巧 2：使用编辑器的查找功能
- 按 `Ctrl+F` 查找缺失的括号
- 按 `Ctrl+H` 快速替换

### 技巧 3：一个错误可能引发多个错误
修复一个问题后，其他错误可能自动消失

### 技巧 4：复制时别忘了粘贴
- 在输出框修改后，记得 📋 复制到剪贴板
- 或直接在输出框修改代码

## 🆘 常见问题

**Q: 为什么有很多错误？**
A: 通常是因为一个大问题（如环境不闭合）导致后续判断全部错误。先修复红色错误。

**Q: 能自动修复吗？**
A: 目前支持手动修复。未来版本会增加"一键修复"功能。

**Q: 为什么没有显示错误？**
A: 恭喜你！说明代码没问题。错误面板只在有问题时显示。

**Q: 怎么关闭错误检查？**
A: 点击错误面板右上角的 ✕ 按钮即可隐藏。

## 📋 检查清单

处理 LaTeX 代码时，按这个顺序检查：

- [ ] 所有 `{` 是否有匹配的 `}`？
- [ ] 所有 `[` 是否有匹配的 `]`？  
- [ ] 所有 `(` 是否有匹配的 `)`？
- [ ] 每个 `\begin{}` 都有 `\end{}`？
- [ ] 所有 `$` 符号成对出现？
- [ ] 中文是否在数学模式外？
- [ ] 是否有拼写错误的命令？

## ⚡ 一分钟快速修复指南

| 错误类型 | 快速修复 |
|---------|---------|
| 🔴 花括号不匹配 | 数一下左右 `{` `}` 的个数，补齐 |
| 🔴 方括号不匹配 | 数一下左右 `[` `]` 的个数，补齐 |
| 🔴 环境不闭合 | 搜索 `\begin{` 在末尾添加对应 `\end{}` |
| 🟡 $ 符号不匹配 | 数 `$` 的个数，应该是偶数 |
| 🟠 中文在数学模式 | 把中文移到 `$` 外面，用 `\text{}` 包装需要的中文 |
